# Evermist

[English](README.md) · **Русский**

Настольное приложение для показа боевых карт D&D на телевизоре с **туманом войны**, который вы рисуете вживую со второго экрана.

Без аккаунтов, без серверов, без подписок — всё работает локально на вашем компьютере. Одно окно смотрит на ваш ноутбук, другое — на телевизор, который видят игроки, и вы открываете подземелье по мере того, как они его исследуют.

> **Чем это *не* является:** полноценным виртуальным столом (VTT). Здесь нет фишек, трекера инициативы и кубиков. Evermist делает одну вещь — показывает карту и скрывает её части — и старается делать это красиво.

## Возможности

- **Два синхронизированных экрана** — окно мастера, которым управляете вы, и чистое окно игрока (без кнопок и курсора) для телевизора.
- **Живописный туман войны** — мягкий, анимированный облачный туман, а не плоский чёрный цвет. Открывайте кистью или рисуйте области показа/скрытия как редактируемые фигуры.
- **Квадратная и гексагональная сетка** — настраиваемые размер, смещение, цвет и прозрачность.
- **Статичные и видео-карты** — перетащите JPG/PNG или анимированную карту MP4/WebM (например, из Dungeon Alchemist).
- **Сцены** — сохраняйте несколько карт и переключайтесь между ними прямо во время игры с плавным переходом.
- **Большие карты** — работает плавно даже на картах 10000×6000.

### Первый запуск

Evermist бесплатен и **не подписан цифровой подписью** (сертификаты подписи стоят денег), поэтому при первом запуске операционная система покажет одноразовое предупреждение безопасности. Это нормально для независимого софта — вот как его обойти:

- **Windows:** если видите *«Система Windows защитила ваш компьютер»*, нажмите **Подробнее → Выполнить в любом случае**.
- **macOS:** если видите *«Evermist нельзя открыть, так как Apple не может проверить…»*, **щёлкните по приложению правой кнопкой → Открыть**, затем нажмите **Открыть** в диалоге. (При двойном клике такой возможности в первый раз не будет.)
- **Linux:** сделайте AppImage исполняемым (`chmod +x Evermist-*.AppImage` или *Свойства → Права → Разрешить выполнение файла как программы*), затем запустите.

Система запомнит выбор, поэтому это происходит лишь один раз.

## С чего начать

После запуска Evermist вы увидите **окно мастера** — это ваш управляющий экран, который игроки никогда не видят.

1. **Откройте окно игрока.** В правой панели в разделе **Player** нажмите **Open Window**. Появится второе окно без кнопок — именно его видят игроки. Перетащите его на телевизор (или второй монитор) и нажмите **Fullscreen**.
2. **Загрузите карту.** Перетащите файл карты (JPG, PNG, MP4 или WebM) прямо в окно мастера. Он загрузится как новая сцена и сразу будет **полностью скрыт туманом**.
3. **Открывайте карту.** Выберите инструмент на верхней панели и открывайте то, что видит группа:
   - **Brush (кисть)** — стирайте туман от руки (размер регулируется клавишами `[` и `]`).
   - **Rectangle / Circle / Polygon (прямоугольник / круг / многоугольник)** — нарисуйте аккуратную область показа, идеально для комнат и коридоров.
   - **Reveal / Shroud (показать / скрыть)** — переключайте, *открывает* фигура область или *снова прячет* (удобно, чтобы закрыть дверь за группой).
   - **Select (V) (выбор)** — двигайте, изменяйте и удаляйте уже нарисованные области.
4. **Подгоните сетку.** В разделе **Grid** правой панели переключайтесь между квадратной и гексагональной сеткой и настраивайте размер/смещение, пока она не совпадёт с сеткой карты.
5. **Выведите на телевизор.** Когда включён **Auto** (в разделе Player), каждое ваше изменение автоматически появляется на экране игрока. Хотите ручной контроль? Выключите Auto и нажимайте **Send ▶** (или клавишу `Space`), когда готовы открыть следующую комнату. **Sync View** подстраивает камеру игрока под вашу.
6. **Переключайте карты во время игры.** Откройте панель **Scenes**, чтобы сохранить несколько карт и переключаться между ними плавным переходом — удобно для перехода из таверны в подземелье без потери атмосферы.

> **Совет:** нажмите `?` в окне мастера в любой момент, чтобы увидеть полный список горячих клавиш.

## Запуск из исходников

Шага сборки нет — это обычный JavaScript в оболочке Electron.

```bash
npm install     # один раз, после клонирования
npm start       # запустить настольное приложение
```

Собрать установщик для вашей платформы:

```bash
npm run build         # Windows portable .exe
npm run build:mac     # macOS .dmg
npm run build:linux   # Linux AppImage
```

Релизы для всех трёх платформ собираются автоматически через GitHub Actions при пуше тега `v*` — см. [`.github/workflows/release.yml`](.github/workflows/release.yml).

## О файлах карт

Медиа карт (`.webm` / `.mp4`) и папка локальных данных приложения **не** входят в этот репозиторий — они хранятся на диске рядом с приложением. Клонирование даёт только код; свои карты добавляйте через приложение.
